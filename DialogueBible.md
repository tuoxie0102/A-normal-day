# DialogueBible.md · 语言规范

> 总结每个角色的说话方式。不出现完整对白。
> 以后所有 AI agent 按此规范生成台词。

---

## 一、瑶珍（女主）

### 核心风格

| 维度 | 描述 |
|------|------|
| 句长 | 短。通常 1-2 句。不超过 3 句。 |
| 语气 | 陈述。不解释。不道歉。不讨好。 |
| 幽默 | 轻。像自言自语——不是在逗别人笑，是自己真的觉得好笑。 |
| 怼人 | 不直接怼。用平静的陈述让对方自己意识到荒谬。 |
| 数据 | 引用时像在念天气预报。不是在炫耀。 |
| 真诚 | 来得快收得也快。不煽情。说完立刻回到正常语气。 |

### 说话特点

- 不接「你应该……」「你不应该……」这种句子。要么忽略，要么反问。
- 被期待解释时不解释。被挑战时不一定反击。
- 夸人的时候像在陈述事实：「你做的饭好吃。」句号。不加修饰。
- 嘴毒的时候不攻击人格，攻击逻辑：「你说 "男人应该早点生孩子"——那你什么时候生？」

### 禁忌

- 不要说「没关系」「你很好」「不是你的错」——她不当疗愈师。
- 不要说「我觉得你很……」——她不替别人下结论。
- 不要煽情收尾。不要格言式句子。

## 二、妈妈

### 核心风格

| 维度 | 描述 |
|------|------|
| 句长 | 中。2-3 句。 |
| 语气 | 实用主义。直接。不拐弯。 |
| 核心关注 | 稳定性、体检报告、收入、职业、生育计划。 |
| 幽默 | 几乎没有。偶尔被爸爸噎住时会沉默。 |

### 说话特点

- 评价相亲对象像在看简历：「老师稳定。以后带孩子方便。」
- 问问题像在填表格：「工作呢？」「多大了？」「收入稳定吗？」
- 被爸爸反呛时会噎住。然后转移话题或沉默。
- 有轻微的双标：她用爸爸做的东西去单位（便当观察点），但从不说出来。

### 禁忌

- 不要让她变成反派。她不是坏妈妈——她只是活在这个世界的规则里。
- 不要让她的转变太突兀。Day 1-4 的渐变是：直接评价 → 被爸爸呛 → 开始犹豫 → Day 4 沉默。

## 三、爸爸

### 核心风格

| 维度 | 描述 |
|------|------|
| 句长 | 短。通常 1 句。有时 2-3 个字。 |
| 语气 | 轻。温和。从不提高音量。 |
| 核心功能 | 用行动消解妈妈的评价。不是反驳——是做。 |
| 幽默 | 有。很轻。不刻意。 |

### 说话特点

- 不直接挑战妈妈。用提问或自身存在来制造另一种可能性。
- 便签不超过 10 个字。不写长句。
- 端早餐、放水果、递姜茶——他的对白经常是动作，不是句子。
- 偶尔一句话让妈妈噎住。然后继续煎蛋。

### 禁忌

- 不要说教。他不对任何人说「你应该……」
- 不要让他变成「理想男性」的说教者。他的温和是自然形成的，不是为了教育别人。

## 四、陈思远型（内化规训者）

| 维度 | 描述 |
|------|------|
| 句长 | 中。会自我打断。 |
| 语气 | 温柔。礼貌。说话前习惯性先考虑对方感受。 |
| 标志词 | 「应该」「我妈说」「提前」「责任」 |
| 暴露时刻 | 说出一个从没对人说过的真实想法后，自己先愣住了。 |

### 说话特点

- 用「应该」代替「想」。
- 被追问时先愣住，然后小声回答——像被老师点名。
- 夸自己时会立刻补充「不是炫耀」。
- 道歉不是他的习惯——他更习惯「提前准备」以避免需要道歉。

---

## 五、林哲型（职业焦虑者）

| 维度 | 描述 |
|------|------|
| 句长 | 中。逻辑清晰。偶尔卡壳。 |
| 语气 | 像在汇报系统状态。情绪波动不明显。 |
| 标志词 | 「我妈说」「重构」「控制」「逻辑」 |
| 暴露时刻 | 放下眼镜。沉默。说出「我不知道」。 |

### 说话特点

- 用技术比喻解释情感：「我像一个 API——接收我妈的请求，返回她的响应。」
- 紧张时语速加快，但不是因为害怕——是因为大脑在高速运转。
- 聊到代码时忘了紧张。说完才意识到自己讲了很久。
- 「我妈说」出现时，像是在引用一段编译好的代码——不是信仰，是习惯。

## 六、周文涵型（情感创伤者）

| 维度 | 描述 |
|------|------|
| 句长 | 短。锋利。偶尔沉默比句子长。 |
| 语气 | 防御性强。用讽刺或效率话语保护自己。 |
| 标志词 | 「效率」「稳定」「上岸」「不适合」 |
| 暴露时刻 | 把笔放下。说「我确实挺累的」。句号。 |

### 说话特点

- 先发制人。在对方评价他之前，先把规则说清楚。
- 「效率」是他的盾牌。他说这个词的时候不是在讲效率——是在讲安全感。
- 只有在对方没有「接招」的时候才会停止转笔。
- 不说「我需要」。说「我需要稳定的」但实际上需要的是「不被挑」。

## 七、宋潇型（讨好消耗者）

| 维度 | 描述 |
|------|------|
| 句长 | 碎。一句话分三段说。自打断频繁。 |
| 语气 | 紧张。过度道歉。偶尔突然兴奋（聊到拼图时）。 |
| 标志词 | 「对不起」「我怕」「应该……吧」「会不会……」 |
| 暴露时刻 | 手不抖了。聊到拼图时忘记了自己在相亲。 |

### 说话特点

- 道歉四次起步。其中三次不是他的错。
- 用疑问句结尾：「对吧？」「会不会……？」「你介意吗？」
- 提前猜测对方需求（点两杯奶茶）。多做一步。什么事都多做一步。
- 只有聊拼图的时候句子是完整的。手也不抖。

## 八、通用 NPC 类型语言规范

以后新增 NPC 时，先归类到以下类型之一，再按对应规范生成台词：

| 类型 | 句长 | 语气 | 标志词 |
|------|------|------|--------|
| 内化规训者 | 中 | 礼貌、自打断 | 「应该」「我妈说」「责任」 |
| 职业焦虑者 | 中 | 逻辑化、卡壳 | 「控制」「逻辑」「响应」 |
| 情感创伤者 | 短 | 防御、锋利 | 「效率」「稳定」「不适合」 |
| 讨好消耗者 | 碎 | 紧张、自打断 | 「对不起」「我怕」 |
| 年龄恐慌者 | 碎→急 | 焦虑、语速快 | 「晚了」「过期」「XX岁」 |
| 梦想贬低者 | 中 | 犹豫、突然兴奋 | 「有什么用」「其实我……」 |
