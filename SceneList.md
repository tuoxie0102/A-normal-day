## 每个场景的详细功能

### S00 · TITLE

| 属性 | 值 |
|------|-----|
| 背景 | 静态图或简单深色底 + 标题字 |
| UI | 「开始游戏」「相亲档案」「退出」 |
| 功能 | 进入新游戏 / 继续上次进度 / 翻阅过往记录 |
| 进入条件 | 游戏启动 |
| 退出条件 | 点击开始 → S01 |

### S01 · HOME

| 属性 | 值 |
|------|-----|
| 背景 | 家中早餐桌。爸爸在厨房。妈妈在餐桌。 |
| 角色 | 爸爸、妈妈、瑶珍（背影或侧影） |
| UI | 电视/手机显示当天新闻。对话自动播放。 |
| 功能 | 展示当天新闻。爸妈对话（3-5句）。便当细节。 |
| 进入条件 | 每天开始（Day 1 从 TITLE 来，Day 2+ 从 RECORD 来） |
| 退出条件 | 对话结束 → 自动切换至当天相亲场景 |

### S02 · CAFE

| 属性 | 值 |
|------|-----|
| 背景 | 咖啡馆窗边。雨天。 |
| NPC | 陈思远 |
| UI | 观察模式（闪烁物品 + 时间条 + 第一印象面板）→ 对话 UI（文本框 + 态度按钮 + 跳过） |
| 功能 | ACT 2-5 完整流程 |
| 进入条件 | S01 结束 → Day 1 |
| 退出条件 | 道别动画结束 → S06 |

### S03 · TEAHOUSE

### S06 · RECORD

| 属性 | 值 |
|------|-----|
| 背景 | 暗色底 + 《今日相亲记录》模板 |
| UI | 结构化记录面板。可滚动。「下一天」按钮。 |
| 功能 | 展示当天相亲结果。第一印象 + 观察 + 关键句 + 瑶珍记录。 |
| 进入条件 | 当天相亲场景结束 |
| 退出条件 | 点击「下一天」→ S01（Day N+1）或一周汇总 |

### S07 · ARCHIVE

| 属性 | 值 |
|------|-----|
| 背景 | 同 S06 或暗色底 |
| UI | 过往记录列表。点击展开详情。 |
| 功能 | 收藏品系统。玩家回翻过往相亲记录。 |
| 进入条件 | TITLE 画面点击「相亲档案」 |
| 退出条件 | 返回 TITLE |
