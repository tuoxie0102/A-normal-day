# 视觉资产清单

> 按场景和类型分类。标注优先级：⭐⭐⭐ = 第一版必须 / ⭐⭐ = 第二版 / ⭐ = 打磨期。
>
> **图片放入 `assets/` 对应子文件夹。每个子文件夹内有 README 说明具体文件命名。**

## 文件夹结构

```
assets/
├── scenes/       ← 场景背景（5张）
├── characters/   ← 角色半身像（6张）
├── objects/      ← 物品特写（12张）
├── ui/           ← UI 切片/设计稿
└── audio/        ← 音效/音乐（Phase 3）
```

### 瑶珍（玩家角色）
- 29 岁女性。自由职业者。自信、直接。
- 视觉：中长发、不刻意打扮但有风格、常带便当袋
- **处理方式**：第一版可能不画瑶珍正面。用第一人称视角（玩家不看到自己）。但可以在早晨场景出现瑶珍的背影/侧影，或在观察阶段出现她伸手拿便当的手部特写。
- 如果画：休闲西装 + 牛仔裤。色彩偏暖（暗红、驼色、米白）。

### 陈思远
- 28 岁男性。小学老师。干净温柔。习惯性自我审视。
- 视觉：浅蓝长袖衬衫（扣到最上面）、指甲整洁、微瘦、微笑时欠身
- 姿势：坐在咖啡馆窗边。手放在温水杯上。稍微前倾。

### 林哲
- 31 岁男性。后端工程师。内向。逻辑清晰。
- 视觉：黑框眼镜、公司文化衫（"代码改变世界"）、头发微乱、带电脑包
- 姿势：坐在茶室木桌前。手指在桌面上轻敲。电脑包放在脚边。

### 周文涵
- 30 岁男性。平面设计师。锐利。话少。
- 视觉：洗到发白的黑色卫衣、右手食指有笔茧、反戴棒球帽或帽衫兜帽、带速写本
- 姿势：坐在公园长椅最右端。手里转笔。不看镜头。

### 宋潇
- 26 岁男性。待业中。敏感。讨好型。手抖。
- 视觉：微瘦、手打毛衣（有点大、袖口有线头）、面前两杯奶茶
- 姿势：坐在奶茶店角落。手在杯子上微微发抖。文件夹从包里戳出。

### 爸爸
- 50+ 岁男性。全职父亲。温和。从不提高音量。
- 视觉：围裙、便当盒、短发、微笑。总是站在厨房里。
- 用于早晨场景。

### 妈妈
- 50+ 岁女性。建筑师。冷静。翻报纸。
- 视觉：西装、短发或梳起、端着咖啡杯。坐在餐桌旁。
- 用于早晨场景。

**风格方向**：半身像或全身像（取决于场景）。不要超级写实。线条 + 平涂色彩。色彩从场景背景中提取，与环境融合。

## 八、制作建议

1. **场景背景**：如果预算/时间有限，用 AI 图像生成打底 + Photoshop 修整。方向关键词：`cozy interior illustration, muted warm palette, flat color, soft shadows, 1920x1080`。

2. **角色**：最难也最重要。建议先定一个角色做完整测试（建议陈思远——他是第一个，且视觉元素最清晰：浅蓝衬衫、温水杯、雨天咖啡馆）。测试通过后，其余三人批量推进。

3. **物品特写**：可以拍照 + 后期处理（降低饱和度、加平光效果）。不需要手绘。

4. **UI**：建议在 prototype.html 里直接 CSS 实现，然后再出设计稿确认。避免"先画再码"的返工。
