# 音频

当前运行时已切换到按场景命名。

## 目录

- `assets/audio/bgm/`：场景 BGM
- `assets/audio/news/`：新闻/广播类音频
- `assets/audio/_legacy/`：旧命名资源备份，运行时不再引用

## 当前命名规则

- 早餐：`bgm_breakfast.m4a`
- 世界/过场：`bgm_world.m4a`
- 咖啡馆：`bgm_cafe.m4a`
- 茶室：`bgm_teahouse.m4a`
- 公园：`bgm_park.m4a`
- 日记：`bgm_diary.m4a`
- 早餐新闻：`news_breakfast.mp3`

运行时音频映射在项目根目录 `audio_map.md` 和 `prototype.html` 中维护。
